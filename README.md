# IT Support & Consulting Company Website

This is a static website for a small-business IT support and consulting company.

## Purpose

This website serves as the online presence for the company, providing information about services, contact details, and establishing credibility with potential clients.

## Structure

```
/
├── index.html          # Main landing page
├── css/
│   └── styles.css      # Stylesheet
├── js/
│   └── script.js       # JavaScript functionality
└── assets/             # Images, logos, and other media
```

## Usage

This is a simple static website that can be:
- Hosted on any web server or static hosting platform (GitHub Pages, Netlify, Vercel, etc.)
- Opened directly in a browser for local development
- Deployed without any build process or dependencies

## Development

To view the site locally:
1. Open `index.html` in a web browser
2. Or use a simple local server: `python -m http.server 8000`

No build tools or package managers are required.
